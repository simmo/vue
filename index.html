Vue

by Mike Simmonds

Vue is a monitor for media queries. It requires no installation, setup or configuration. Follow the steps below to get going:

Note: Can only identifiy CSS loaded on the same domain.

  1. Add link to bookmarks: Vue
  2. Go to the site you wish to test & open the JS console.
  3. Click the bookmark!
